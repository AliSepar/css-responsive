# CSS responsive exercise
Practicing with @media query changing website style in different view size.

### Using @media query 
```
    @media only screen and (max-width:300px){
        body{
        background-color: red;
        }
        /* to show only small div 
        #medium{
            display: none;
        }
        #large{
            display: none;
        } */
        div{ 
            font-size: 14px;
        }

    }    
```
